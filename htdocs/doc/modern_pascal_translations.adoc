:description: Translations of "Modern Object Pascal Introduction for Programmers"

Thanks to many kind people, this book has been translated to several languages:

. https://castle-engine.io/modern_pascal_introduction_bg.html[Bulgarian] - thanks to https://github.com/alpinistbg[Юлиян Иванов (alpinistbg)]

. https://castle-engine.io/modern_pascal_introduction_chinese.pdf[Chinese] (WIP) - thanks to https://github.com/robsean[郑 (robsean)]

. https://castle-engine.io/modern_pascal_introduction_russian.html[Russian] - thanks to https://github.com/SpanerRh[Alexander (SpanerRh)] and https://github.com/eugeneloza[Eugene Loza]

. link:modern_pascal_spanish[Spanish] - thanks to https://jorgeturiel.es/[Jorge Turiel Fernández]

. https://castle-engine.io/modern_pascal_introduction_ukrainian.html[Ukrainian] (WIP) - thanks to https://github.com/iso4free[Vadim Vitomsky (iso4free)]

Most of the translations linked above are hosted within the https://github.com/modern-pascal/modern-pascal-introduction/[GitHub repository of the book]. If you'd like to improve them, or propose a translation to a new language, go ahead -- it is simplest to just fork the repository and send a pull request. If anything is unclear, just ask on https://castle-engine.io/talk.php[Castle Game Engine Forum, Discord or any other channel].

**Big thank you to everyone who contributed to this book and translations!** Let's continue to show the world how Pascal https://castle-engine.io/why_pascal[is a productive and modern programming language].
