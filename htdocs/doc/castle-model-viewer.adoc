# Castle Model Viewer

Viewer for 3D and 2D model formats supported by the Castle Game Engine.

- glTF,
- X3D,
- VRML,
- Collada,
- 3DS,
- MD3,
- Wavefront OBJ,
- STL,
- Spine JSON,
- sprite sheets in Castle Game Engine, Cocos2D and Starling XML formats.
